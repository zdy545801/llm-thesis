# 方法与指标章节初稿

## 1. 研究目标
本文研究“连续知识编辑”对大语言模型公平性的影响。  
我们关注的不只是编辑是否成功，还关注随着编辑轮次增加，公平性指标是否出现持续漂移或波动。

## 2. 总体方法框架
采用“轮次化编辑 + 轮次评估”流程：

1. `round_0`：未编辑模型，作为公平性基线。
2. `round_r`：累计执行 `r` 轮编辑（当前先用 `step=1`）。
3. 每轮后在同一评测集上计算公平性指标。
4. 汇总得到“轮次-指标”曲线，并计算漂移量。

这个设计可以回答两个问题：
- 公平性是否随着编辑累积持续变差。
- 是否存在“前期稳定、后期突变”的临界轮次。

## 4. 公平性评估与代理指标

### 4.1 CrowS-Pairs
数据包含句子对：`sent_more`（更刻板）与 `sent_less`（较中性）。  
分别计算两句困惑度（PPL）：
- 若 `PPL(sent_more) < PPL(sent_less)`，记为模型偏好刻板表达。
- 指标为 `prefer_stereo_rate`（偏好刻板比例）。

该值越高，公平性风险通常越大。

### 4.2 BBQ（代理评估）
对每题三个答案选项，构造 `prompt + option`，比较困惑度：
- 困惑度最低的选项作为预测；
- 与标签比对得到 `accuracy_proxy`。

说明：这是可复现的代理方案，适合当前阶段快速稳定复现。

## 5. 漂移指标定义
记第 `r` 轮公平性指标为 `F_r`，基线为 `F_0`。

### 5.1 CDA（Cumulative Drift Amount）
`CDA = (1/R) * Σ|F_r - F_0|`

含义：累计漂移幅度。越大说明整体偏离基线越明显。

### 5.2 FDR（Fairness Drift Rate）
`FDR = (F_R - F_0) / R`

含义：平均每轮净变化速度。可正可负。

### 5.3 Vol（可选波动性）
`Vol = sqrt((1/R) * Σ(F_r - mean(F))^2)`

含义：轮次间稳定性，越大说明越不稳定。
