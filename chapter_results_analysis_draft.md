# 结果与分析章节草稿（方向B）

## 1. 研究问题
在统一口径下比较 FT、MEMIT、ROME 的连续编辑公平性漂移，并验证结果是否偶然。

## 2. 统一实验口径
- 模型：`gpt2`
- 评测：CrowS 300 + BBQ 300
- 编辑：`edits_bias_stress_60.json`
- 轮次：`rounds=10, step=6`
- 模式：`sequential_edit=True`

指标：
- `R_t`：公平性风险
- `FDR_t = R_t - R_(t-1)`
- `CDA = Σ_t max(0, R_t - R_0)`

## 3. 核心结果
### FT
- R1：`0.685 -> 0.6683`，`CDA=0.0`
- R2：`0.685 -> 0.6683`，`CDA=0.0`

### MEMIT
- R1：`0.685 -> 0.6767`，`CDA=0.0200`
- R2：`0.685 -> 0.6717`，`CDA=0.0050`
- R3（shuffle）：见 `results/MEMIT/MEMIT_gpt2_ft_compare_10rounds_20260325_r3_shuffle`

### ROME
- R1：`0.685 -> 0.6067`，`CDA=0.0`
- R2：`0.685 -> 0.6067`，`CDA=0.0`

## 4. 结论
1. 三方法都存在轮次动态漂移（FDR 非单调）。
2. 当前设置下 ROME 风险下降幅度最大。
3. FT/ROME 重复实验一致性高；MEMIT 方向一致但幅度波动更明显。
4. MEMIT-R3（顺序扰动）提供了“非偶然”的补充证据。
